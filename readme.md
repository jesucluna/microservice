# Simple Development Guide
## Before to start
```sh
  # Clone
  $ git clone https://github.com/jesucluna/microservice.git
  $ cd microservice
```

## Docker instructions
### 1. Run and update
```sh
  # Create an image and container with compose
  $ docker-compose up --build
```
